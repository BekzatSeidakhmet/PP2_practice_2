# Music Player (Pygame)

## 📌 Description
A simple music player built with Pygame that allows you to control playback using keyboard keys.

## ⚙️ Features
- Play, Stop, Next, Previous controls
- Playlist support
- Displays current track
- Keyboard-based interaction

## 🛠 Technologies
- Python
- Pygame (mixer module)

## ▶️ How to Run
```bash
python main.py
🎮 Controls
P → Play
S → Stop
N → Next track
B → Previous track
Q → Quit
