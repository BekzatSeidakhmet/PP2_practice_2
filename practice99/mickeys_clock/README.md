# Mickey's Clock (Pygame)

## 📌 Description
This project is a digital-style clock built with Pygame.  
It displays the current system time using rotating clock hands inspired by Mickey Mouse.

## ⚙️ Features
- Displays **minutes and seconds**
- Real-time synchronization with system clock
- Rotating clock hands using images
- Updates every second

## 🛠 Technologies
- Python
- Pygame

python main.py
