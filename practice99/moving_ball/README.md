# Moving Ball Game (Pygame)

## 📌 Description
A simple interactive game where a red ball moves across the screen using arrow keys.

## ⚙️ Features
- Smooth movement using keyboard input
- Boundary checking (ball cannot leave screen)
- Simple graphics using Pygame

## 🛠 Technologies
- Python
- Pygame

## ▶️ How to Run
```bash
python main.py
🎮 Controls
Arrow Keys → Move ball
Close window → Exit
