# Python RegEx Exercises

This repository contains solutions to 10 basic Python Regular Expression (RegEx) exercises.  
Each task demonstrates how to use Python's `re` module for pattern matching and string manipulation.

---

## 🧠 Key RegEx Symbols Used

| Symbol | Meaning |
|--------|----------|
| `.` | Any character |
| `*` | 0 or more |
| `+` | 1 or more |
| `{n,m}` | Between n and m repetitions |
| `[]` | Character set |
| `()` | Grouping |
| `\1` | Backreference |
